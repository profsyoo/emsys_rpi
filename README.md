# Embedded Systems Examples (Raspberry Pi 5)

이 저장소는 Embedded Systems 수업에서 사용하는
Raspberry Pi 5 기반 예제 코드 모음입니다.

## 사용 방법
```bash
git clone https://github.com/profsyoo/emsys_rpi.git
cd emsys_rpi
```

## Reference
- lgpio: https://abyz.me.uk/lg/index.html
- rpi-lgpio(for the dprecated rpi-gpio API compatibility): https://rpi-lgpio.readthedocs.io/en/latest/
- gpio-zero: https://gpiozero.readthedocs.io/en/stable/
